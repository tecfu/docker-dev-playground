# docker-dev-playground

## A Docker image full of REPLS and programming languages

See [https://hub.docker.com/r/tecfu/docker-dev-playground/](https://hub.docker.com/r/tecfu/docker-dev-playground/)

- C# [via mono]

```
$ csharp
```

- DotNet

```
$ dotnet
```

- Java [via Java 9]

```
$ jshell
```

- Julia

```
$ julia
```

- Nodejs

```
$ node
```

- Python

``` 
$ python
```

- R

```
$ R
```
